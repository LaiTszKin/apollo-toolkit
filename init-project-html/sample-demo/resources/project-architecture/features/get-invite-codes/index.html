Feature module · get-invite-codes

獲取邀請碼

對外能力：已登入使用者請求系統生成新的邀請碼（生產邀請碼）。

使用者故事

使用者進入「我的邀請碼」頁面 → 按生成 → 系統檢查額度與帳號狀態 → 產生密碼學隨機字串並寫入 invite_codes。

跨子模組交互（含 generator 的多次重試呼叫、DB UNIQUE 衝突回傳、跨功能 invite_codes 行被「邀請碼註冊」消費）已在宏觀架構圖內統一呈現，本頁不重述。
