Feature module · invite-code-registration

邀請碼註冊

對外能力：訪客憑邀請碼建立帳號（消費邀請碼）。

使用者故事

訪客提交註冊 → 系統以邀請碼狀態判定 → 通過則建立帳號並核銷邀請碼，失敗則不留下任何持久化副作用。

跨子模組交互（誰呼叫誰、誰讀誰寫、回應如何流回）以及與「獲取邀請碼」功能的資料行傳遞，已在宏觀架構圖內統一呈現，本頁不重述。

子模組自身職責頁面（自身 I/O + 內部資料流）
