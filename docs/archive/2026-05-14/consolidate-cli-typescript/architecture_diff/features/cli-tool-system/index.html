CLI 工具系統

Submodules

  • cli-dispatcher Service

    CLI 參數解析、安裝/解除安裝流程、互動式 UI、幫助文本 (TypeScript)

  • tool-runner Service

    工具註冊表、handler 直接呼叫 + spawn fallback 雙模式執行

  • installer Service

    技能安裝/解除安裝、manifest 管理 (不再複製 scripts/)

  • tool-implementations Service

    所有工具 handler 的 TypeScript 實現 (lib/tools/)

  • shared-utils Pure fn

    共享純函數: 終端顏色、格式化、型別定義
