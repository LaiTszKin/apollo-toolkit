# apltk codegraph

`apltk codegraph` is a local codebase exploration tool powered by `@colbymchenry/codegraph`. Use it to inspect real symbols, files, call relationships, and change impact before making design or implementation decisions.

Before using CodeGraph in a workflow, run:

```bash
apltk codegraph --help
apltk codegraph <subcommand> --help
```

Choose the subcommands from the live help output instead of relying on memorized command lists. Treat CodeGraph results as code evidence for agent judgment, not as an automatic architecture modeller or spec verifier.
