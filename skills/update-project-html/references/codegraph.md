# apltk codegraph

`apltk codegraph` is a local codebase exploration tool powered by `@colbymchenry/codegraph`. Use it to inspect real symbols, files, call relationships, and change impact before updating architecture documentation.

Before using CodeGraph in a workflow, run:

```bash
apltk codegraph --help
apltk codegraph <subcommand> --help
```

Choose the subcommands from the live help output instead of relying on memorized command lists. Architecture atlas updates still require explicit evidence mapping and design judgment.
